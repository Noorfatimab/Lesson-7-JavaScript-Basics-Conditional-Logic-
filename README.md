# Lesson 7 – JavaScript Basics (Conditional Logic)

This project introduces conditional logic in JavaScript.

## Features
- Paragraph with id `displayText` that changes dynamically
- Button `clickMeBtn` updates text with a message
- Button `decisionBtn` uses `if/else` to choose between two messages
- Demonstrates decision-making using variables and conditions

## Files included
- index.html – Webpage structure
- script.js – JavaScript logic
- README.md – Project explanation
